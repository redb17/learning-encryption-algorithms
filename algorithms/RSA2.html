The Pros of RSA:

  1. Messages can be shared seretly without any prior physical meeting.
  2. It provides the Authentication of the two parties.

The Cons of RSA:

  1. The key pair generation is slow and thus RSA is not preferred to be used as ephemeral keys but rather static keys i.e. a key pair is used over a long time.
  2. Since RSA is used as static keys, it does not provide Forward Secrecy. An attacker can keep collecting the messages until he gets the private keys and then can decrypt all the prior messages.
