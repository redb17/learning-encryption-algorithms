Cryptography of a message requires the following:

  1. Generation of encryption and decryption keys without any physical meeting between the sender and receiver.
  2. Forward Secrecy i.e. to generate new keys for each session so that old messages cannot be decrypted if attackers find the present keys.
  3. Authentication of the sender and receiver and prevent any man-in-the-middle to act as the receiver (for the actual sender) and sender (for the actual receiver).
